# News-Headline-Web-Scraper

This project is a simple **Python web scraper** that collects the latest **BBC News headlines** and saves them into a `.txt` file.

---

##  Features
- Scrapes top headlines from **BBC News**
- Uses `requests` to fetch website HTML
- Uses `BeautifulSoup` to parse and extract headlines
- Saves all headlines into a file named `bbc_headlines.txt`
- Simple and beginner-friendly Python script

---

##  Technologies Used
- Python
- Requests
- BeautifulSoup (bs4)
- HTML parsing
